指令菜单

（不再维护 CMD 操作，请尽量不要使用指令）
命令 参数 用途
xy 无 开启 / 关闭显示坐标
xy_mode 无 切换显示坐标的方式，可以设置为显示标尺或直接在方块上显示坐标。（直接在方块上显示坐标可以获得更快的渲染速度）
cls 无 清屏
坐标x 坐标y 无 在某位置放置红石粉，如果那个位置已有物品，则清除那个物品
坐标x 坐标y 物品 (朝向) 无 在某位置放置某物品，并对已有物品进行覆盖。部分方块可设置朝向，留空则默认向上。
坐标x 坐标y 朝向 无 设置某位置的物品的朝向（针对红石中继器）
. 坐标x 坐标y 点击某位置的开关（如拉杆，按钮）
resize 宽 高 重设地图大小
save 无 保存地图为项目文件
reset_map_offset 无 将地图画面的xy偏移都回到0
up 偏移量 将地图往上偏移n个单位，一单位是10像素
down 偏移量 将地图往下偏移n个单位，一单位是10像素
left 偏移量 将地图往左偏移n个单位，一单位是10像素
right 偏移量 将地图往右偏移n个单位，一单位是10像素
zoom 缩放大小 将地图缩放至多少百分比
zoom+ 缩放大小 在地图原先的缩放基础上，再放大多少个百分比
zoom- 缩放大小 在地图原先的缩放基础上，再减小多少个百分比
get_map_offset 无 得到地图画面的xy偏移量
get_map_zoom 无 得到地图画面的缩放大小
import 坐标x 坐标y 导入一个项目到当前项目，并指定子项目位于当前项目的左上角位置
line（指令头可省略） 坐标x1 坐标y1 坐标x2 坐标y2 (物品)

画红石直线（不能弯曲），如果在末尾加上物品名表示以某方块填充直线。

指令头( line )可以省略，即可以直接输入所有参数。
clear 坐标x1 坐标y1 坐标x2 坐标y2 清除 (x1, y1) 到 (x2, y2) 的所有方块，即以空气方块覆盖。
exit 无 退出程序。
cmd_window_top 无 使cmd窗口顶置。
cmd_window_no_top 无 使cmd窗口不顶置。
map_window_top 无 使地图显示窗口顶置。
map_window_no_top 无 使地图显示窗口不顶置。
resize_cmd_window 窗口宽 窗口高 设置cmd窗口大小（单位为cmd窗口大小单位）
resize_map_window 窗口宽 窗口高 设置地图显示窗口大小（单位：像素）

 

物品名称表

英文代号 中文名称
null 空气方块
powder 红石粉
rod 拉杆
button 按钮
torche 红石火把（仅可由中继器充能）
light 红石灯
relay 红石中继器（可以使电路单向导通，为红石火把充能，但不支持其它特性）
cross 交叉线（竖直方向和水平方向的线路不会连接到一起）

 

朝向表

英文代号 中文名称
up 向上
down 向下
left 向左
right 向右

更新日志

版本 时间 更新内容
内测版 Ver 0.1 2020.10.4 初步实现程序
内测版 Ver 0.2 2020.10.4 更新交叉排线板，新增项目导入功能
内测版 Ver 0.3 2020.10.5

扩大图形界面窗口大小，命令行窗口大小，顶置命令行，使操作更方便。

新增绘制红石直线，清除区块的命令，优化算法。
Ver 1.0 2020.10.5 程序各项功能测试通过。
Ver 1.1 2020.10.7

项目名由MinecraftRedstoneVirtualBox改为MinecraftRedstoneSimulate。

帮助文档改为单独的网页。

交叉排线板的贴图更新。

更新坐标显示方式为标尺，且添加了网格线；增加xy_mode命令，可以切换坐标显示模式。

新增程序icon，设置了命令行和图形界面的窗口标题。

可以直接将rsp地图文件拖动到程序中来打开地图。
Ver 1.2 2020.10.11

项目名由MinecraftRedstoneSimulate改为MinecraftRedstoneSimulator。

优化绘图算法，加快渲染速度。（在Ver 1.1中，用标尺方式渲染一个200*200的地图需要21s，优化后需要3s，如果使用直接在方块上显示坐标的方法可以得到更快的渲染速度(<1s)）
Ver 1.3 2020.10.24

更新按钮贴图。

修复使用zoom指令缩放后地图显示范围未改变的bug。

修复在某种条件下CROSS方块信号断路的bug。

更新line指令，可以省略指令头。

新增下列指令：
exit
cmd_window_top
cmd_window_no_top
map_window_top
map_window_no_top
resize_cmd_window
resize_map_window

修复打开项目后help指令无效的问题。

修复打开较大项目失败的问题（即内存泄漏问题）。
Ver 1.4 2020.10.25

更新交叉排线板贴图为根据充能状态动态生成贴图。

修复指令处理部分内存泄漏隐患。
Ver 2.0
 2022.08.04 对模拟器进行了全面更新，支持鼠标交互操作，优化算法和绘制机制，极大地提升了运行速度，支持了更多的红石电路。
Ver 2.1
 2022.08.22 支持红石脉冲，更新地图格式，更新地图包
